
### Install

Install dependencies.

```bash
yarn install
```

Serve with hot reload at <http://localhost:5173>.

```bash
yarn dev
```

### Lint

```bash
yarn lint
```

### Typecheck

```bash
yarn typecheck
```

### Build

```bash
yarn build
```

### Test

```bash
yarn test
```

View and interact with your tests via UI.

```bash
yarn test:ui
```
