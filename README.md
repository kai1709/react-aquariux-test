
### Install

Install dependencies.

```bash
yarn install
```

Serve with hot reload at <http://localhost:5173>.

```bash
yarn dev
```

### Lint

```bash
yarn lint
```

### Typecheck

```bash
yarn typecheck
```

### Build

```bash
yarn build
```

### Enviroment
Please provide these ENV variables to .env
- VITE_API_KEY
- VITE_CRYPTO_KEY
